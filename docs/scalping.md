激进Scalping的核心逻辑
目标：捕捉微小价格波动，快速进出，避免隔夜风险和资金费率。
基本决策循环（每1-5秒或每根1m K线运行一次）：
推荐信号组合（激进版，适合BTC/ETH/SOL等高流动性币种）：

时间框架：1分钟或更短（甚至tick级别，但API限频要小心）。
常见指标组合：
EMA交叉 + RSI：快EMA（5/9）上穿慢EMA（21）且RSI(7或14) > 50 → 开多；反之开空。
布林带突破 + 成交量：价格突破上轨 + 量放大 → 开多；下轨 → 开空。
动量突破：价格突破最近5-10根K线高点 + MACD柱状图正向放大。
订单簿辅助（高级）：bid/ask深度不平衡时顺势入场。

入场过滤：只在高波动时段交易（如美盘或亚洲开盘重叠），避免低波动震荡期（用ATR判断波动率>阈值才交易）。
出场：目标利润0.3%-0.8%，止损0.2%-0.5%（盈亏比至少1.5:1以上）。用移动止损（Trailing Stop） 或时间止损（持仓超过2-5分钟强制平）。

杠杆与仓位（激进但要活下来）：

杠杆：20x-50x（甚至更高，视币种波动），但单笔风险严格控制在账户总资金的0.5%-1%。
仓位计算：仓位大小 = (账户资金 × 风险比例) / (止损距离 × 合约乘数)。
逐仓模式必开（Isolate），避免一个仓位爆了连累全部。

手续费与资金费率影响（针对快进快出）：

资金费率：几乎可以忽略！因为持仓时间极短，通常不会跨过OKX的8小时（或4小时）结算点。即使跨，也只影响极少数仓位。
最大成本是手续费（Maker/Taker费率）。OKX VIP等级越高费率越低，激进策略要追求吃单（Taker）多，优先选择低费率账户或返佣。
回测时必须把双边手续费（开+平）精确扣除，否则收益看起来很好，实盘直接被吃光。

2. 扩展逻辑（让策略更激进、更稳定）

多时间框架过滤：1m信号 + 5m趋势确认（避免逆大势）。
动态参数：用ATR自适应止损距离（波动大时止损/目标都放大）。
订单高级用法（OKX强项）：
Post-Only（只挂Maker单，省手续费）。
Iceberg / TWAP（大仓位拆分，避免滑点）。
Trigger订单（计划委托，突破时自动开）。

机器学习轻量版：用简单模型预测下一根K线方向概率，只在>70%时交易。
并行多币种：同时跑BTC、ETH、SOL等高流动性对，分散机会但控制总风险。
风控升级（激进策略的救命稻草）：
日/小时最大亏损上限（亏2%全平仓休息）。
连续亏损笔数限制（连亏5笔暂停交易）。
整体杠杆上限 + 强平价格实时监控。
黑天鹅过滤：重大新闻时暂停（可集成外部API）。
