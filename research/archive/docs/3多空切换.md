OKX 多空切换 + 趋势过滤 + AI调参策略规范（纯文本）
	1.	数据输入
K线数据：OHLCV（5m / 15m / 1h / 4h）
手续费：maker / taker
滑点：固定或比例
资金费率（Funding Rate）
	2.	指标计算
MA200（趋势判断）
RSI（超买超卖）
布林带（回调/反弹）
成交量均值（量能过滤）
	3.	市场状态判定
if price_4h > MA200_4h and slope(MA200_4h) > 0:
state = “bull”
elif price_4h < MA200_4h and slope(MA200_4h) < 0:
state = “bear”
else:
state = “range”

约束：
state == “range” → 不交易
	4.	交易逻辑

牛市（只做多）
条件：
price_15m > MA200_15m
RSI < oversold_threshold 或触及布林下轨

执行：开多

平仓：
RSI > overbought_threshold 或触及布林上轨

熊市（只做空）
条件：
price_15m < MA200_15m
RSI > overbought_threshold（反弹）或触及布林上轨

执行：开空

平仓：
RSI < oversold_threshold 或触及布林下轨
	5.	交易限制
最小交易间隔 >= N 根K线
每日最大交易次数 <= M
连续亏损 >= K 次 → 暂停交易
	6.	成本模型
实际成交价 = 市场价 * (1 ± slippage)
每笔收益 = 毛收益 - 手续费 - 滑点
	7.	资金费率过滤
if funding_rate > 0:
优先做空
elif funding_rate < 0:
优先做多
	8.	风控
止损：1% ~ 3%
止盈：>= 止损 * 1.5
最大回撤限制（如20%）
	9.	AI调参参数空间
MA周期：100 ~ 300
RSI周期：5 ~ 30
RSI阈值：20 ~ 80
布林带窗口：10 ~ 50
布林带倍数：1 ~ 4
交易间隔：5 ~ 50
止损：0.5% ~ 5%
止盈倍数：1 ~ 3
	10.	优化目标函数
score = sharpe_ratio - α * max_drawdown - β * trade_count
或
score = total_return - 回撤惩罚
	11.	回测规范
训练集 / 测试集分离
必须包含手续费、滑点、资金费率
多周期验证（5m / 15m / 1h）
	12.	输出结果
最优参数
总收益
夏普比率
最大回撤
交易次数
胜率
盈亏比
	13.	执行流程
加载数据 → 计算指标 → 判断市场状态 → 执行交易 → 记录结果 → 计算指标 → AI调参 → 输出结果
	14.	核心约束
震荡市场不交易
必须加入成本模型
禁止全历史调参
必须限制交易频率
